This folder contains files that are no longer actively used in the project but are kept for reference.

- **legal.min.css** - Old minified CSS for legal pages (replaced by legal.fixed.css)
- **legal.min.js** - Old minified JavaScript for legal pages (replaced by legal.fixed.js)

### `screenshots/`
Documentation and testing reference images:
- Privacy page screenshots (various sections and viewports)
- Home page sections (hero, stay, experiences, contact, gallery)
- Final design screenshots

### `config_backups/`
- **package.json.bak** - Backup of package.json
- **vite.config.ts.unused** - Unused Vite configuration file
