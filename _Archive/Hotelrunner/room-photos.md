## Photo Download Instructions

To download Hotelrunner photos for Beds24 upload:

```bash
# Download Safari Tent photo
curl -o safari-tent.jpg "https://cdn-cms2.hotelrunner.com/assets/photos/large/5c9c1be0-5891-496f-a2c8-74efd59e85d8.jpg"

# Download Comfort Tent photo
curl -o comfort-tent.jpg "https://cdn-cms1.hotelrunner.com/assets/photos/large/bafaeaa4-a3d3-4f07-ae0f-2ec64c74897a.jpg"

# Download Garden Cottage photo
curl -o garden-cottage.jpg "https://cdn-cms1.hotelrunner.com/assets/photos/large/8c13e90f-11c2-4da1-99c5-83a9ef4495b0.jpg"
```

## Image Requirements for Beds24

- **Minimum width:** 1200px (recommended 1920px for best quality)
- **Format:** JPG, PNG
- **Aspect ratio:** 4:3 or 16:9 preferred
- **File size:** Under 5MB per image
- **Naming convention:** Use descriptive names (e.g., safari-tent-exterior.jpg)
